# 要求定義

## 目的

Windowsデスクトップ右下に控えめに常駐し、ユーザーが日々のToDoを素早く確認・追加・完了できる軽量タスク管理アプリを提供する。

## 想定利用

- 複数ユーザーへの配布
- 業務利用
- 個人ごとのローカルタスク管理
- 作業の邪魔にならない控えめな表示

## 対象環境

- Windows 10
- Windows 11
- .NET Framework 4.8

## 主要機能

- タスク追加
- タスク削除
- 完了/未完了切り替え
- 任意の期限日
- 期限切れ/今日期限の表示
- SQLite保存
- 右下ウィジェット表示
- タスクトレイ常駐
- Windows起動時の自動起動設定

## 非機能要件

- 管理者権限なしで利用できること
- ユーザーごとにデータと設定を保存すること
- 外部通信を行わないこと
- アプリ更新時にユーザーデータを消さないこと
- 常駐時の負荷を抑えること
